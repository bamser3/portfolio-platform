# portfolio-platform

## Features

### Landing Page with Project List:
Displays all added projects in a clean, card-based layout with placeholder images.

### Add Project Form:
Users can add new projects with a name and description (images use a placeholder by default).

### Search Functionality:
Filter projects dynamically by project name or description.

## Installation

### Clone the repository

```bash
git clone https://github.com/bamser3/portfolio-platform.git
cd portfolio-platform
```
### Install dependencies
```bash
npm install
```
### Start the development server
```bash
npm run dev
```
### Open the app in your browser
The link should be similar to
```bash
http://localhost:5173
```
